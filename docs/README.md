# Gradex

> A modern online learning platform for university students.

---

# Overview

Gradex is an educational platform that enables university students to purchase and access high-quality video courses.

The platform provides:

- Course catalog
- Secure authentication
- Online payments
- Video streaming
- Progress tracking
- Instructor dashboard
- Admin dashboard

---

# Vision

Create the best online learning experience for university students by delivering high-quality educational content through a fast, secure, and scalable platform.

---

# Target Users

- Students
- Instructors
- Administrators
